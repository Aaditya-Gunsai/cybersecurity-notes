## Writeup Structure

Each writeup follows this format:

- **Vulnerability explanation** — what it is and why it exists
- **Step-by-step exploitation** — how the attack works in practice
- **Payloads used** — actual payloads with notes
- **Key observations** — what I learned and what to watch for
